# Supply-Chain-Operations-Analytics  
Supply Chain Operations | Inventory Risk Management | SQL Analytics | Power BI Dashboard | Python Automation  

---

## 📌 Overview  
End-to-end supply chain analytics project to improve **inventory optimization, demand planning, and operational efficiency** using SQL, Python, and Power BI.

---

## 🎯 Problem Statement  
Supply chains face:
- Stockouts and overstocking  
- Poor demand forecasting accuracy  
- Low operational visibility  

---

##  Tools & Technologies  
- SQL  
- Python (Pandas, NumPy)  
- Power BI  
- Excel  

---

##  Key KPIs  
- Inventory Turnover  
- Order Fulfillment Rate  
- Demand vs Supply Analysis  
- Stockouts / Overstock Levels  

---

##  Key Insights  
- Identified high-risk SKUs contributing to stockouts  
- Detected inventory imbalance patterns across products  
- Improved visibility into demand trends for planning  

---

##  Output  
Interactive Power BI dashboard for KPI monitoring, trend analysis, and decision-making.

---

##  Business Impact  
- Improved inventory planning efficiency  
- Reduced operational inefficiencies  
- Enabled data-driven decision-making
